## 6. 工具授权校验

```java
checkAuthorization(toolCode, context) {
    // 1. 获取工具定义
    tool = AiToolDefinitionRepository.findByToolCode(toolCode);

    // 2. 查询授权记录
    auths = AiToolAuthorizationRepository
        .findByToolIdAndEnabled(tool.getId());

    // 3. 遍历授权匹配
    for (auth : auths) {
        switch (auth.authType) {
            case AGENT:
                if (context.agentId == auth.authValue) {
                    // 检查 dataScope
                    return checkDataScope(context, auth);
                }
            case SCENE:
                if (context.sceneType == auth.authValue) {
                    return checkDataScope(context, auth);
                }
            case USER:
                if (context.userId == auth.authValue) {
                    return checkDataScope(context, auth);
                }
        }
    }

    // 4. 无匹配授权 → 拒绝
    throw BusinessException(TOOL_NOT_AUTHORIZED);
}
```

| 字段 | 说明 |
|------|------|
| `id` | 主键 |
| `tool_code` | 工具编码 |
| `call_type` | 调用类型：INTERNAL/MCP |
| `session_id` | 会话ID（可选） |

| `tool_result_json` | 调用结果JSON |

| `exec_duration_ms` | 执行耗时（毫秒） |
| `error_message` | 错误信息 |
| `created_at` | 创建时间 |
