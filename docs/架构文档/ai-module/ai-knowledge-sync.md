## 4. 知识源类型定义

| 枚举值 | 说明 | 可入库条件 |
|--------|------|-----------|
| `PUBLIC_ARTICLE` | 公开文章 | 文章状态=公开 且 用户可访问 |
| `FORUM_POST` | 论坛帖子 | 帖子状态=正常 且 用户可访问 |
| `AUTHOR_PROFILE` | 作者公开资料 | 作者已认证 |
| `MANUAL_ENTRY` | 后台手动维护 | 后台手动创建 |

## 6. 同步任务管理

### 6.1 触发同步

```
POST /api/sys/ai/knowledge/entry/sync
│
├─ sourceType: PUBLIC_ARTICLE / FORUM_POST / ...
├─ entryIds: [1, 2, 3] (可选，空表示全量)
└─ forceReprocess: true/false (可选)
         │
         ▼
    ┌─────────────────────────┐
    │ 1. 创建同步任务记录      │
    │    status = PENDING     │
    └─────────────────────────┘
         │

    ┌─────────────────────────┐
    │ 2. 异步执行同步          │
    │    processSyncTask()    │
    └─────────────────────────┘
         │

    ┌─────────────────────────┐
    │ 3. 更新任务状态          │
    │    PENDING → RUNNING     │
    └─────────────────────────┘
         │
```

## 7. 事件监听器实现

```java
@Component
@RequiredArgsConstructor
public class KnowledgeSyncContentListener {

    private final AiKnowledgeEntryRepository entryRepository;

    @EventListener
    public void onContentChange(ContentChangeEvent event) {
        // 1. 校验知识源是否启用
        if (!isKnowledgeSourceEnabled(event.getSourceType())) {
            return;
        }

        // 2. 校验内容是否可入库
        if (!isContentAccessible(event)) {
            return;
        }

        // 3. 根据变更类型处理
        switch (event.getChangeType()) {
            case PUBLISHED:
                createEntry(event);
                break;
            case UPDATED:
                markOutdated(event);
                break;
            case HIDDEN:
            case DELETED:
                markDisabledOrDeleted(event);
                break;
        }
    }

    private boolean isContentAccessible(ContentChangeEvent event) {
        // 公开内容 + 非白名单 = 可入库
        return event.isPublic() && !event.isWhitelist();
    }
}
```
