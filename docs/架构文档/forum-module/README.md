## 快速导航

### 开发者视角

1. **理解模块架构** → 参考 [00-forum-module-overview.md](./00-forum-module-overview.md)
2. **查看数据表结构** → 参考 [forum-data-model.md](./forum-data-model.md)
3. **理解核心业务流程** → 参考 [forum-core-flow.md](./forum-core-flow.md)

### 运营视角

1. **管理版块** → 后台 `/api/sys/forum/sections`
2. **治理帖子** → 后台 `/api/sys/forum/posts`
3. **治理回复** → 后台 `/api/sys/forum/replies`
4. **公开浏览** → `GET /api/forum/*`
5. **用户操作** → `GET/POST /api/user/forum/*`

### 数据结构

| 实体 | 说明 |
|------|------|
| `ForumSection` | 论坛版块 |
| `ForumPost` | 论坛帖子 |
| `ForumReply` | 论坛回复 |
| `ForumPostChannelLink` | 帖子与频道关联（分享到聊天频道） |
