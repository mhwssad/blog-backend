## 子域划分

| 子域 | 定位 | 核心职责 |
|------|------|----------|
| `collection` | 收藏管理 | 用户收藏夹维护、文章收藏、收藏计数 |
| `comment` | 评论管理 | 评论发表、点赞、回复树、评论审核 |
| `taxonomy` | 分类标签 | 文章分类树、标签维护 |
| `footprint` | 用户足迹 | 浏览历史记录 |
| `interaction` | 互动记录 | 点赞等互动行为管理 |
| `friendlink` | 友情链接 | 前台友链展示、后台友链管理 |
| `shared` | 共享服务 | 公共内容查询服务、模型转换器 |
