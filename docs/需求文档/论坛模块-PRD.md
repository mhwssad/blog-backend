## 2. 模块定位

论坛模块负责承接平台的长期讨论沉淀能力。博客偏作者内容生产，聊天频道偏实时互动，论坛位于二者之间，负责结构化话题、帖子回复、社区讨论留存和后续知识沉淀。

当前代码已具备论坛版块、帖子、回复基础模型和 Repository，并已完成二期 P0 的公开侧、用户侧和帖子频道分享主链路。后续目标是继续补齐后台侧、举报通知和更完整频道联动的端到端闭环。

## 3. 模块目标

- 建立论坛版块、帖子、回复的完整业务链路。
- 支持用户围绕话题进行长期讨论。
- 支持公开浏览、登录互动和作者 / 管理员治理。
- 与频道保持职责分离，并支持帖子手动挂接频道讨论。
- 接入举报、通知、文件生命周期和后台治理体系。
- 为 AI 知识库 / RAG 提供可控的公开论坛知识来源。

## 4. 目标用户

- 访客：浏览公开版块、公开帖子和部分回复。
- 注册用户：发帖、回复、点赞、收藏、举报。
- 作者：参与高质量内容沉淀，可拥有更高配额或展示权益。
- 管理员：管理版块、帖子、回复、置顶、精华和违规内容。
- 超级管理员：处理跨模块异常和高风险治理动作。

## 5. 第二期范围

论坛第二期能力包括：

- 论坛版块公开查询和后台管理。
- 用户发帖、编辑、删除。
- 用户回复、编辑、删除。
- 帖子公开列表、详情和回复列表。
- 帖子点赞、收藏。
- 帖子置顶、精华、隐藏、删除。
- 回复隐藏、删除。
- 论坛帖子和回复举报。
- 回复、点赞、加精等通知事件。
- 帖子手动挂接频道，并支持双向跳转。

## 7. 核心功能需求

### 7.1 版块

版块需支持：

- 版块列表。
- 版块详情。
- 版块排序。
- 版块状态：启用、停用。
- 版块可见范围。
- 版块发帖等级限制。
- 后台创建、编辑、启用、停用。

### 7.2 帖子

帖子需支持：

- 用户创建帖子。
- 用户编辑自己的帖子。
- 用户删除自己的帖子。
- 公开帖子分页。
- 公开帖子详情。
- 按版块筛选。
- 按关键词搜索。
- 按 latest / hot / top 排序。
- 帖子状态：正常、隐藏、删除、锁定。
- 帖子可见范围。
- 帖子统计：浏览数、回复数、点赞数、收藏数。

### 7.3 回复

回复需支持：

- 用户回复帖子。
- 用户回复指定回复。
- 用户编辑自己的回复。
- 用户删除自己的回复。
- 回复分页或楼层查询。
- 回复状态：正常、隐藏、删除。
- 回复楼层号。
- 父回复统计同步。

### 7.4 互动

论坛互动需支持：

- 点赞帖子。
- 取消点赞帖子。
- 收藏帖子。
- 取消收藏帖子。
- 查询当前用户点赞和收藏状态。
- 互动必须与帖子状态、可见范围和用户权限联动。

### 7.5 后台治理

后台需支持：

- 版块管理。
- 帖子分页筛选。
- 帖子隐藏 / 恢复。
- 帖子删除。
- 帖子置顶 / 取消置顶。
- 帖子加精 / 取消加精。
- 回复隐藏 / 恢复。
- 回复删除。
- 治理动作写入审计日志。

### 7.6 举报接入

论坛需接入统一举报体系：

- 举报论坛帖子。
- 举报论坛回复。
- 举报处理时可隐藏帖子或回复。
- 举报处理时可选择禁言、封禁或仅记录不处罚。
- 举报结果通知遵守治理模块统一策略。

### 7.7 通知接入

论坛通知至少包括：

- 回复帖子通知帖子作者。
- 回复回复通知被回复人。
- 点赞帖子通知帖子作者。
- 帖子被设为精华通知作者。
- 通知投递前遵守用户通知设置。

### 7.8 频道联动

论坛与频道联动规则：

- 论坛帖子可手动挂接到一个频道。
- 频道消息可展示帖子摘要并跳转帖子详情。
- 帖子详情可展示关联频道入口。
- 帖子隐藏或删除后，频道侧应展示失效状态或禁止继续跳转。
- 论坛和频道不共用分类 / 标签。

## 8. 权限规则

- 访客可查看公开版块和公开帖子。
- 访客不可发帖、回复、点赞、收藏。
- 登录用户可在满足等级、状态和可见范围规则时发帖和回复。
- 用户只能编辑或删除自己的帖子和回复。
- 管理员可治理授权范围内的帖子和回复。
- 超级管理员可处理跨状态和异常数据。
- 被禁言用户不得发帖或回复。

## 9. 数据一致性

- 发帖后需同步版块帖子数。
- 回复后需同步帖子回复数和父回复回复数。
- 点赞 / 收藏后需同步帖子统计字段。
- 帖子隐藏、删除后需影响公开列表、详情、举报处理和频道跳转。
- 多表联动必须在统一事务边界内完成。

## 10. 非目标

第二期论坛不重点处理：

- 复杂推荐算法。
- 频道消息自动沉淀为帖子。
- 论坛和频道共用分类 / 标签。
- 多级复杂版主管理体系。
- 完整勋章和积分商城。

## 11. 验收标准

- 用户可完成发帖、编辑、删除、回复、点赞、收藏。
- 公开侧可查看版块、帖子和回复。
- 管理员可完成版块管理和帖子 / 回复治理。
- 论坛帖子和回复可被举报并进入后台处理。
- 论坛通知能覆盖回复、点赞和加精等核心事件。
- 帖子可手动挂接频道，并支持双向跳转。
- 被隐藏或删除内容不会在公开侧继续正常展示。
