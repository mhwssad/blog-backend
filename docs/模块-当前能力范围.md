## 一、AI 模块

### 当前已支持

**用户侧能力**

| 能力        | 说明                                                              |
| ----------- | ----------------------------------------------------------------- |
| AI 会话管理 | 创建会话、查询会话列表、查询会话详情、关闭会话                    |
| 消息收发    | 发送消息并接收同步完整响应，支持消息历史分页查询，含 RAG 引用来源 |
| 消息角色    | 支持 `user` / `assistant` / `system` 三种角色类型                 |
| 响应状态    | 支持 `success` / `failed`，失败时记录错误信息                     |
| 配额查询    | 用户可查询个人每日 AI 配额及剩余额度                              |
| Token 计算  | 记录每次调用的请求 Token、响应 Token、总 Token 及额度消耗         |
| Agent 任务  | 用户可发起 Agent 任务、查询任务列表、查询任务详情、取消任务       |

**后台管理能力**

| 能力           | 说明                                                                                           |
| -------------- | ---------------------------------------------------------------------------------------------- |
| 渠道配置管理   | 多渠道配置管理，支持渠道的创建、更新、启停、删除，含 API Base URL、API Key、模型参数、配额配置 |
| 会话查询       | 后台分页查询用户会话列表、查看会话详情（含渠道信息、模型信息）                                 |
| 使用统计       | AI 调用日志分页查询，使用统计聚合（总调用次数、成功次数、失败次数、总额度消耗）                |
| Agent 定义管理 | 后台创建、更新、启停、删除 Agent 定义                                                          |
| Agent 任务管理 | 后台分页查询 Agent 任务、查看任务详情                                                          |
| 知识源配置     | 后台查询、更新知识源配置，切换启停状态                                                         |
| 知识条目管理   | 后台分页查询知识条目、查看详情、更新状态                                                       |
| 知识同步任务   | 后台触发知识同步任务，分页查询同步任务列表、查看详情，重试失败任务                             |
| 工具定义管理   | 后台创建、更新、启停、删除工具定义                                                             |
| 工具授权管理   | 后台创建、更新、删除工具授权配置                                                               |
| 工具执行测试   | 后台可对工具进行执行测试，查看调用日志                                                         |
| MCP 服务管理   | 后台创建、更新、启停、删除 MCP 服务配置，发现 MCP 工具，查看工具快照，查询连接健康状态         |

**核心链路校验**

| 校验项     | 说明                                 |
| ---------- | ------------------------------------ |
| 全局开关   | 全局 AI 功能开关校验，关闭时拒绝服务 |
| 额度校验   | 平台每日额度校验、用户每日额度校验   |
| 上下文长度 | 上下文长度超限校验                   |
| 会话归属   | 无权操作他入会话                     |
| 会话状态   | 禁止向已关闭会话发送消息             |

### 后续推荐加

| 能力项           | 说明                                                               |
| ---------------- | ------------------------------------------------------------------ |
| 流式/SSE 响应    | 当前为同步非流式响应，建议新增流式输出能力以提升体验               |
| 后台多维统计     | 当前仅支持分页查询，建议新增会话数、消息数、用户数等多维度统计聚合 |
| RAG 知识检索优化 | 当前已支持基础 RAG 引用来源展示，可考虑增强检索精度和展示效果      |
| Agent 执行日志   | 当前仅记录任务状态，建议完善执行过程详细日志                       |
| MCP 工具自动同步 | 当前需要手动触发发现，建议优化为自动发现与增量同步机制             |
| AI 对话体验优化  | 支持对话标题自动生成、会话置顶、消息已读未读状态                   |

## 二、聊天模块

### 当前已支持

**用户侧能力**

| 能力     | 说明                                                                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 会话管理 | 分页查询我的会话、查询会话详情、打开或创建单聊会话、加入公开频道或公开群、离开频道或公开群                                                       |
| 消息收发 | 分页查询会话消息、发送文本消息、发送文件消息                                                                                                     |
| 消息操作 | 编辑消息、撤回消息、删除我的消息视图（仅当前用户视角）                                                                                           |
| 已读推进 | 推进会话已读游标                                                                                                                                 |
| 群聊管理 | 创建群聊、查询群聊详情、查询群成员、邀请群成员、设置群管理员、取消群管理员、转让群主、设置群成员禁言、更新群公告、移除群成员、退出群聊、解散群聊 |

**后台管理能力**

| 能力     | 说明                                             |
| -------- | ------------------------------------------------ |
| 会话查询 | 后台分页查询会话、查询会话详情、查询会话成员     |
| 消息审计 | 分页查询会话消息、查询消息详情、分页查询消息回执 |
| 成员管理 | 更新成员角色、更新成员状态、更新成员禁言         |
| 消息管理 | 后台撤回消息                                     |
| 会话管理 | 更新会话状态                                     |

**核心链路校验**

| 校验项   | 说明                                        |
| -------- | ------------------------------------------- |
| 频控     | 用户级分钟频控                              |
| 敏感词   | 敏感词拦截                                  |
| 媒体任务 | 媒体任务完成后的 `message_updated` 更新推送 |

### 后续推荐加

| 能力项            | 说明                                   |
| ----------------- | -------------------------------------- |
| 富文本引用块      | 当前不支持富文本引用块消息             |
| 跨会话引用        | 当前不支持跨会话引用与多层嵌套回复展示 |
| 审核工作流        | 当前无完整审核工作流和人工复核         |
| 未读/会话缓存     | 当前未实现未读数缓存和会话状态缓存     |
| 消息置顶/已读未读 | 用户侧消息置顶、已读未读状态展示       |
