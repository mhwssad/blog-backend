# 后台页面设计

本文档涵盖 17 个后台管理页面，基于后端 DTO 字段定义详细描述每个页面的表格列、搜索条件、操作按钮、表单字段及对应 API。所有管理页面遵循统一的后台管理框架布局。

**通用布局**：
- 顶部导航栏（Logo、用户信息、退出登录）
- 左侧菜单栏（动态渲染菜单树，基于权限控制可见性）
- 右侧主内容区（页面标题 + 面包屑 + 内容区域）

---

## 3.1 仪表盘

**路由建议**：`/admin/dashboard`

**所需权限**：无特殊权限要求（登录即可访问）

**页面说明**：
仪表盘为后台首页，展示系统核心统计数据。

**统计卡片**：

| 指标 | 说明 | 数据来源 |
|---|---|---|
| 文章总数 | 已发布文章数 | 统计接口 |
| 用户总数 | 注册用户数 | 统计接口 |
| 评论总数 | 评论数 | 统计接口 |
| 今日访问量 | PV/UV | 统计接口 |

**建议组件拆分**：
- DashboardPage — 页面容器
- StatCard — 统计卡片
- TrendChart — 趋势图表（可选）
- RecentActivity — 最近活动列表（可选）
