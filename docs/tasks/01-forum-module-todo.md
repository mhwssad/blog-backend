# 论坛正式模块任务清单

本文档用于从零规划第二期论坛正式模块。论坛负责沉淀长期讨论内容，频道负责实时互动，二者通过帖子挂接频道形成联动。

## 1. 当前状态

**当前阶段：待开始。**

```
可复用基线:
  ✅ 用户、作者、管理员和超级管理员权限体系
  ✅ 博客文章、评论、附件、举报和通知能力
  ✅ 主题频道、频道成员、频道消息和帖子频道轻量挂接结构

待建设:
  ⏳ 论坛版块 / 帖子 / 回复 / 治理 / 搜索 / 通知完整闭环
```

## 2. P0 基础数据模型

- [x] 设计论坛版块表：名称、简介、排序、可见范围、发帖等级、状态。
- [x] 设计论坛帖子表：标题、内容、作者、版块、状态、可见范围、置顶、精华、统计字段。
- [x] 设计论坛回复表：帖子、父回复、回复人、内容、状态、楼层、统计字段。
- [x] 设计帖子附件引用关系，复用 file 模块生命周期。
- [x] 设计帖子与频道关系，复用或升级 `forum_post_channel_link`。
- [x] 定义帖子状态：草稿、已发布、审核中、已拒绝、已删除。
- [x] 定义回复状态：正常、隐藏、删除、审核中。

## 3. P0 用户侧能力

- [ ] 用户创建帖子。
- [ ] 用户编辑自己的帖子。
- [ ] 用户删除自己的帖子。
- [ ] 用户查询自己的帖子列表。
- [ ] 用户发表回复。
- [ ] 用户编辑或删除自己的回复。
- [ ] 用户收藏、点赞帖子。
- [ ] 用户分享帖子到频道。

## 4. P0 公开能力

- [ ] 公开查询版块列表。
- [ ] 公开查询帖子列表。
- [ ] 公开查询帖子详情。
- [ ] 公开查询帖子回复列表。
- [ ] 支持关键词、版块、作者、时间和热度筛选。
- [ ] 访客权限与登录用户权限分层。

## 5. P1 后台治理

- [ ] 后台版块管理。
- [ ] 后台帖子分页、详情、隐藏、恢复、删除。
- [ ] 后台回复分页、隐藏、恢复、删除。
- [ ] 后台帖子置顶、取消置顶、设为精华、取消精华。
- [ ] 接入举报对象类型：论坛帖子、论坛回复。
- [ ] 治理动作写入审计日志。

## 6. P1 联动与通知

- [ ] 回复帖子通知作者。
- [ ] 回复被回复人通知。
- [ ] 点赞帖子通知作者。
- [ ] 帖子被设为精华通知作者。
- [ ] 帖子分享到频道后生成频道消息摘要。
- [ ] 帖子被删除或隐藏后，频道挂接摘要可展示失效状态。

## 7. 测试与文档

- [ ] 补齐论坛 Service 正常路径测试。
- [ ] 补齐越权编辑、越权删除、隐藏内容访问等异常测试。
- [ ] 补齐后台权限测试。
- [ ] 更新 `docs/api文档`。
- [ ] 如新增表结构，更新原始建表脚本。

## 8. 完成标志

- 用户能完成发帖、回帖、浏览和频道分享。
- 管理员能完成版块、帖子和回复治理。
- 论坛帖子可与主题频道形成双向跳转。
- 论坛能力不会绕过内容可见性、举报、通知和文件生命周期规则。
