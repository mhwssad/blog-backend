# AI 知识库、RAG 与 Agents 任务清单

本文档用于规划第二期 AI 增强。第一期 AI 已完成基础问答、渠道配置、额度控制和调用统计，第二期在此基础上扩展知识库 / RAG / agents。

## 1. 当前状态

**当前阶段：待开始。**

```
可复用基线:
  ✅ AI 会话、消息、渠道、额度、调用日志
  ✅ AI 数据读取范围枚举
  ✅ AI 默认禁止读取用户私聊内容
  ✅ 后台 AI 配置与审计

待建设:
  ⏳ 知识源管理
  ⏳ RAG 检索增强
  ⏳ agents 定义与任务执行
```

## 2. P0 知识源边界

- [ ] 明确可入库知识源：公开文章、作者公开资料、论坛帖子、后台维护知识条目。
- [ ] 明确不可入库内容：用户私聊、私密文章、白名单文章、被删除 / 隐藏内容。
- [ ] 设计知识源类型枚举。
- [ ] 设计知识条目表：来源类型、来源 ID、标题、摘要、内容快照、状态、版本。
- [ ] 设计知识同步任务表：任务类型、状态、错误信息、重试次数。
- [ ] 后台配置每类知识源是否启用。

## 3. P0 RAG 主链路

- [ ] 接入向量化模型配置。
- [ ] 实现知识条目分段。
- [ ] 实现向量索引写入和更新。
- [ ] 实现按问题检索相关片段。
- [ ] 将检索结果拼接到 AI 上下文。
- [ ] 回答返回引用来源。
- [ ] 调用日志记录是否启用 RAG、检索耗时和命中数量。

## 4. P1 知识库管理

- [ ] 后台查看知识条目列表。
- [ ] 后台查看知识条目详情。
- [ ] 后台手动触发重建索引。
- [ ] 后台禁用指定知识条目。
- [ ] 内容更新后自动标记知识条目待同步。
- [ ] 内容删除或隐藏后自动失效知识条目。

## 5. P1 Agents

- [ ] 设计 agent 定义表：名称、描述、提示词、启用状态、可用工具、数据范围。
- [ ] 设计 agent 任务表：用户、agent、状态、输入、输出、错误信息。
- [ ] 支持后台创建、编辑、启用、停用 agent。
- [ ] 支持用户发起 agent 任务。
- [ ] 支持任务状态查询。
- [ ] 支持任务执行日志。
- [ ] agent 完成后接入通知中心。

## 6. 权限与安全

- [ ] agent 数据读取范围必须复用 AI 数据范围配置。
- [ ] 高风险 agent 配置变更写审计日志。
- [ ] RAG 不得返回用户无权访问的内容引用。
- [ ] 私聊数据读取继续默认关闭。
- [ ] 额度消耗规则覆盖普通 AI 问答、RAG 问答和 agent 任务。

## 7. 测试与文档

- [ ] 补齐知识源同步测试。
- [ ] 补齐私密内容不入库测试。
- [ ] 补齐 RAG 命中和无命中测试。
- [ ] 补齐 agent 任务状态流转测试。
- [ ] 更新 `docs/api文档/ai-api.md`。

## 8. 完成标志

- AI 能基于公开知识源回答并返回引用。
- 后台可控制知识源和 RAG 开关。
- 用户可发起基础 agent 任务并查询状态。
- 数据权限、额度和审计边界生效。
