# AI 知识库、RAG 与 Agents 任务清单

本文档用于规划第二期 AI 增强。第一期 AI 已完成基础问答、渠道配置、额度控制和调用统计，第二期在此基础上扩展知识库 / RAG / agents。

## 1. 当前状态

**当前阶段：P0 知识源边界已完成，RAG 暂缓，下一步推进 P1 Agents。**

```
可复用基线:
  ✅ AI 会话、消息、渠道、额度、调用日志
  ✅ AI 数据读取范围枚举
  ✅ AI 默认禁止读取用户私聊内容
  ✅ 后台 AI 配置与审计

已完成:
  ✅ 知识源边界定义与枚举（可入库/不可入库）
  ✅ 知识源配置表、知识条目表、同步任务表
  ✅ 知识源配置后台管理（查询、更新、启停）
  ✅ 知识条目后台管理（分页查询、详情、状态变更）
  ✅ 同步任务后台管理（触发、重试、查询）

暂缓:
  ⏸ RAG 检索增强（向量化、分段、检索、上下文拼接）
     原因：P0 阶段优先完成 agents 主线，RAG 作为后续增强独立推进

待建设:
  ⏳ agents 定义与任务执行
```

## 2. P0 知识源边界

- [x] 明确可入库知识源：公开文章、作者公开资料、论坛帖子、后台维护知识条目。
- [x] 明确不可入库内容：用户私聊、私密文章、白名单文章、被删除 / 隐藏内容。
- [x] 设计知识源类型枚举。
- [x] 设计知识条目表：来源类型、来源 ID、标题、摘要、内容快照、状态、版本。
- [x] 设计知识同步任务表：任务类型、状态、错误信息、重试次数。
- [x] 后台配置每类知识源是否启用。

## 6. 权限与安全

- [ ] agent 数据读取范围必须复用 AI 数据范围配置。
- [ ] 高风险 agent 配置变更写审计日志。
- [ ] RAG 不得返回用户无权访问的内容引用。
- [ ] 私聊数据读取继续默认关闭。
- [ ] 额度消耗规则覆盖普通 AI 问答、RAG 问答和 agent 任务。
