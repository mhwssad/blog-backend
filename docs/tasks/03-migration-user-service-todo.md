## 3. P1 外部博客迁移设计

- [x] 梳理外部博客字段来源：v1 固定 JSON 文件，顶层 `sourcePlatform + posts`。
- [x] 建立文章字段映射表：迁移项转换为 `ArticleSaveRequest`，复用后台文章创建链路。
- [x] 建立作者映射策略：v1 由创建任务时指定站内 `authorId`，不做外部作者映射。
- [x] 建立分类和标签映射策略：分类按 `categoryCodes` 匹配文章分类编码且启用，标签按 `tagNames` 匹配名称，不自动创建。
- [x] 建立附件下载、上传和引用绑定策略：先下载外部附件入库并替换正文/封面 URL，再由文章创建链路同步附件引用。
- [x] 明确重复导入幂等键：`sourcePlatform + ":" + externalPostId`，同任务重复预检失败，全局已成功导入则执行时跳过。
- [x] 明确失败回滚和人工修复策略：单篇附件下载或文章创建失败时记录失败，不创建部分文章；失败记录可导出。
