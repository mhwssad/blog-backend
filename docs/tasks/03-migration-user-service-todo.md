# 外部博客迁移与用户自服务任务清单

本文档用于规划第二期迁移工具和用户自服务能力。迁移工具服务于后续外部博客导入，用户自服务接口用于补齐前端账号页面联调缺口。

## 1. 当前状态

**当前阶段：待开始。**

```
可复用基线:
  ✅ 文章、分类、标签、附件、作者和用户体系
  ✅ 文件上传与附件引用
  ✅ 认证、邮箱验证码和密码加密

待建设:
  ⏳ 外部博客导入映射和校验
  ⏳ 用户资料、密码和找回密码自服务接口
```

## 2. P0 用户自服务

- [ ] 用户查询个人资料详情。
- [ ] 用户更新昵称、头像、简介、个人站点等公开资料。
- [ ] 用户修改密码。
- [ ] 找回密码发送邮箱验证码。
- [ ] 找回密码重置密码。
- [ ] 用户搜索接口，支持前端全局搜索用户 Tab。
- [ ] 参数校验使用注解或自定义校验注解，Controller 不写业务校验逻辑。

## 3. P1 外部博客迁移设计

- [ ] 梳理外部博客字段来源。
- [ ] 建立文章字段映射表。
- [ ] 建立作者映射策略。
- [ ] 建立分类和标签映射策略。
- [ ] 建立附件下载、上传和引用绑定策略。
- [ ] 明确重复导入幂等键。
- [ ] 明确失败回滚和人工修复策略。

## 4. P1 迁移工具实现

- [ ] 支持导入任务创建。
- [ ] 支持上传或读取迁移数据文件。
- [ ] 支持预检并返回错误明细。
- [ ] 支持执行导入。
- [ ] 支持导入进度查询。
- [ ] 支持导入结果统计。
- [ ] 支持失败记录导出。

## 5. P2 友情链接

- [ ] 评估友情链接是否进入后台配置。
- [ ] 如进入，实现后台友情链接 CRUD。
- [ ] 实现公开友情链接查询。
- [ ] 支持排序、状态和图标。

## 6. 测试与文档

- [ ] 补齐用户资料更新测试。
- [ ] 补齐修改密码和找回密码测试。
- [ ] 补齐用户搜索权限和分页测试。
- [ ] 补齐迁移预检、幂等和失败回滚测试。
- [ ] 更新认证 / 用户相关 API 文档。
- [ ] 如新增表结构，更新原始建表脚本。

## 7. 完成标志

- 前端账号资料、修改密码、找回密码和用户搜索可完成联调。
- 外部博客迁移至少完成预检和小批量导入闭环。
- 迁移过程可追踪、可重试、可核对。
