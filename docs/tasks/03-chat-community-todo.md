## 3. 场景模型收口

- [x] 使用 `chat_conversation.scene_type` 区分 `single`、`group`、`global_lobby`、`topic_channel`。
- [x] 使用 `visibility_scope` 和 `allow_guest_view` 控制访客可见性。
- [x] 使用 `require_join_to_speak` 控制是否加入后才可发言。
- [x] 使用 `join_rule` 控制 `open`、`approval`、`invite_only`、`private`。
- [x] 使用 `speak_level_limit` 控制发言等级门槛。
- [x] 使用 `member_limit` 控制群 / 频道人数上限。
- [x] 使用 `announcement`、`slow_mode_seconds`、`display_sort`、`channel_category_code` 支撑公告、慢速、排序和分类。

## 4. T06 大厅频道落地

### 4.1 目标

- 大厅频道作为世界频道入口。
- 访客可看不可发。
- 登录用户满足禁言、慢速、等级规则后可发言。
- 管理员可进行大厅治理。

### 4.2 初始化与查询

- [x] 确保唯一大厅会话存在，可启动初始化或懒创建。
- [x] 大厅会话 `scene_type = global_lobby`。
- [x] 大厅允许访客查看历史消息。
- [x] 大厅消息查询支持访客模式，不写入成员和游标。
- [x] 登录用户进入大厅时可自动补建成员和游标。

### 4.3 发言规则

- [x] 未登录用户发送消息被拦截。
- [x] 被禁言用户发送消息被拦截。
- [x] 未达到 `speak_level_limit` 的用户被拦截。
- [x] 慢速模式下同一用户发言间隔不足被拦截。
- [x] 频控和敏感词继续复用现有治理服务。

### 4.4 治理能力

- [x] 管理员设置大厅公告。
- [x] 管理员置顶大厅消息。
- [x] 管理员删除 / 撤回大厅消息。
- [x] 管理员踢出或禁言大厅用户。
- [x] 管理员设置慢速模式和发言等级限制。

### 4.5 测试

- [x] 访客可查看大厅消息。
- [x] 访客发送消息被拦截。
- [x] 低等级用户发送消息被拦截。
- [x] 慢速模式命中时发送被拦截。
- [x] 管理员公告、置顶、撤回、禁言动作生效。
- [x] WebSocket 推送仍能同步大厅消息和治理事件。

## 5. T07 主题频道与频道申请

### 5.1 目标

- 主题频道由管理员创建。
- 用户可申请创建频道。
- 频道加入后才可发言。
- 访客不可查看主题频道消息。

### 5.2 频道管理

- [x] 管理端创建主题频道。
- [x] 管理端编辑频道名称、简介、分类、公告、排序、可见范围。
- [x] 管理端启用 / 禁用频道。
- [x] 管理端设置加入规则和发言等级。
- [x] 管理端查看频道成员和消息。

### 5.3 频道申请

- [x] 使用 `chat_channel_create_application` 保存申请。
- [x] 用户提交频道创建申请。
- [x] 管理员分页查看申请。
- [x] 管理员审核通过后创建频道。
- [x] 管理员拒绝后记录原因。
- [x] 已存在待处理申请时限制重复提交。

### 5.4 成员与发言

- [x] 用户主动加入公开频道。
- [x] 需要审批的频道进入申请流程。
- [x] 非成员不可发言。
- [x] 访客不可查看主题频道消息。
- [x] 被踢出或禁言用户不可发言。

### 5.5 测试

- [x] 管理员创建频道成功。
- [x] 用户申请创建频道成功。
- [x] 申请通过后频道生成。
- [x] 申请拒绝后状态和原因可查。
- [x] 非成员发言被拦截。
- [x] 访客查看主题频道消息被拦截。

## 6. T08 群聊增强与入群方式闭环

### 6.1 目标

- 任何登录用户可建群，但受等级和数量限制。
- 普通群有人数上限。
- 支持公开、私密、审批、邀请链接等入群方式。
- 支持群分类和搜索。

### 6.2 建群规则

- [x] 建群前校验用户等级。
- [x] 建群前校验用户已有群数量上限。
- [x] 创建群时设置群分类、群简介、成员上限、加入规则。
- [x] 创建者自动成为群主。
- [x] 群人数达到上限后拒绝新成员加入。

### 6.3 入群方式

- [x] 邀请后直接入群。
- [x] 私密群需要群主或管理员审批。
- [x] 支持主动申请入群。
- [x] 支持邀请链接入群。
- [x] 邀请链接需要有效期和可选使用次数限制。
- [x] 使用 `chat_group_join_application` 保存入群申请。

### 6.4 群治理

- [x] 群主可设置 / 取消管理员。
- [x] 群主和管理员可处理入群申请。
- [x] 群主和管理员可禁言成员。
- [x] 群主和管理员可更新群公告。
- [x] 群主和管理员可移除成员。
- [x] 管理员不可操作群主和同级管理员。

### 6.5 群搜索

- [x] 按关键词搜索公开群。
- [x] 按分类筛选群。
- [x] 私密群是否展示按 `visibility_scope` 控制。
- [x] 已禁用或已解散群不进入用户搜索结果。

### 6.6 测试

- [x] 等级不足时建群被拦截。
- [x] 达到建群数量上限时建群被拦截。
- [x] 达到人数上限时加入被拦截。
- [x] 私密群主动申请后等待审批。
- [x] 邀请链接过期后不可入群。
- [x] 群管理员越权审批或踢人被拦截。

## 7. T15 论坛与频道挂接接口

### 7.1 目标

- 第一阶段先做帖子手动分享到频道。
- 帖子与频道支持双向跳转。
- 不做频道消息自动沉淀成帖子。

### 7.2 数据与接口

- [x] 使用 `forum_post_channel_link` 保存帖子与频道关系。
- [x] 一个帖子第一阶段只能挂一个频道。
- [x] 分享帖子到频道时生成一条特殊类型消息。
- [x] 频道消息 payload 中保存帖子摘要和跳转信息。
- [x] 论坛帖子页可查询关联频道。
- [x] 频道消息可跳转论坛帖子详情。

### 7.3 权限

- [x] 明确谁可以分享帖子到频道。
- [x] 分享目标频道必须允许当前用户发言或具备管理权限。
- [x] 已删除、未公开或无权限访问的帖子不可分享。
- [x] 频道禁言用户不可分享帖子。

### 7.4 测试

- [x] 帖子分享至频道成功。
- [x] 同一帖子重复挂接同一频道保持幂等。
- [x] 同一帖子挂接第二个频道被拦截。
- [x] 无权限帖子分享被拦截。
- [x] 频道消息中的帖子摘要可正常返回。

- 入群申请、邀请链接和审批链路完整可测。
- 论坛挂接保持轻量手动分享，不引入自动沉淀。
